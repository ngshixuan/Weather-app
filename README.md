# 🌤️ Weather App

A simple and responsive **Weather Application** built with **JavaScript** that fetches real-time weather data using the **AccuWeather API**.  
Users can search for a city and instantly view current weather conditions.

This project demonstrates how to work with **REST APIs**, **asynchronous JavaScript**, and **dynamic DOM manipulation**.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Display current temperature  
- ☁️ Show weather condition and icon  
- 🌍 Real-time weather data from AccuWeather  
- ⚡ Fast and lightweight vanilla JavaScript app  
- 📱 Simple and responsive UI  

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **AccuWeather API**

## ⚙️ How It Works

1. User enters a city name.
2. JavaScript sends a request to the **AccuWeather API**.
3. The API returns the latest weather data.
4. The app dynamically updates the UI with:

- Temperature
- Weather condition
- Weather icon
- City name
